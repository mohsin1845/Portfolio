# AKSHAT Photography Portfolio

## Project Overview
Cinematic photography portfolio website inspired by Peter McKinnon. Features dark luxurious aesthetic, fullscreen sections, and premium storytelling layout.

## Tech Stack
- Next.js 16.2.6
- React 19.2.4
- Tailwind CSS v4
- Framer Motion (animations)
- Supabase (database)
- TypeScript

## Sections
1. **Hero Landing** - Fullscreen with parallax, large "AKSHAT" typography
2. **About Me** - Photographer bio with stats and elegant image frame
3. **Nature Gallery** - Masonry grid with hover zoom effects
4. **Kashmir Mountains** - Full-width with overlay text (Featured series)
5. **Wedding Portfolio** - Split layout with image grid
6. **Business Promotion** - Corporate branding showcase
7. **Instagram/Reels** - Social media stats and video previews
8. **Contact/Booking** - Form with project type selection

## Design System

### Color Theme: Dark Luxury Gold
- Background: `#0a0a0a` (deep black)
- Primary text: `#fafafa` (off-white)
- Accent: `#d4af37` (gold)
- Secondary: `#b8962e` (darker gold)
- Glass effect: `rgba(255, 255, 255, 0.05)` with backdrop blur

### CSS Classes
- `.glass` - Glassmorphism background
- `.glass-card` - Gradient glass card for content
- `.text-gradient` - Gold gradient text
- `.hover-zoom` - Image zoom on hover (transform: scale 1.08)

### Animations
- `FadeIn` component - Scroll-triggered fade up animations
- Parallax hero effect using `useScroll` and `useTransform`
- Smooth 0.6s cubic-bezier transitions on images

## Customization

### Adding Sections
1. Add new section in `page.tsx` with unique `id`
2. Add nav link in `layout.tsx` navbar array
3. Use `<FadeIn>` wrapper for scroll animations

### Changing Images
Images in `public/`. To update:
1. Download new image to `public/`
2. Update `src` prop in Image component

### Color Theme
Edit CSS variables in `globals.css`:
```css
:root {
  --background: #0a0a0a;
  --foreground: #fafafa;
  --accent: #d4af37;
  --accent-secondary: #b8962e;
}
```

### Animations
The `FadeIn` component handles scroll animations:
```tsx
<FadeIn>
  <YourContent />
</FadeIn>

// With delay
<FadeIn delay={0.2}>
  <YourContent />
</FadeIn>
```

### Image Gallery
Uses CSS grid with tall image support:
```tsx
<div className="image-gallery-grid">
  <ImageCard src="..." className="image-tall" /> // Spans 2 rows
  <ImageCard src="..." />
</div>
```

## Supabase Integration

### Database
- **Project URL**: https://aawvflvkirgzhvmlduzm.supabase.co
- **Table**: `contacts`

### Contacts Table Schema
```sql
CREATE TABLE contacts (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  name TEXT NOT NULL,
  email TEXT NOT NULL,
  project_type TEXT,
  message TEXT,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);
```

### Form Submission
The contact form in page.tsx uses the `handleSubmit` function to:
1. Collect form data (name, email, project_type, message)
2. Insert into Supabase `contacts` table
3. Show loading state while submitting
4. Display success/error message after submission
